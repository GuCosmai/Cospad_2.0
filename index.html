Manual do CosPad

Bem-vindo ao CosPad — seu estúdio de música na web. Aqui vai um guia rápido:

Pads

  • Adicionar Pad: clique em "Adicionar Pad" na coluna Opções.
  • Editar Pad: clique no ícone ✎ no canto do pad para mudar nome, tecla, cor, áudio e volume.
  • Tocar: clique no pad ou pressione a tecla atribuída.
  • Arrastar áudio: arraste um item da Biblioteca para um pad para atribuir o áudio.

Biblioteca de Áudio

  • Arraste arquivos do seu PC para a área "Arraste áudios aqui" ou use o botão para selecionar arquivos.
  • Você pode pré-ouvir (▶) ou remover arquivos da biblioteca.
  • Ao exportar projeto, áudios locais podem ser incluídos como dataURLs (arquivo grande).

Faixas

  • Adicionar Faixa: botão "Adicionar Faixa" na seção Faixas.
  • Arraste áudios da Biblioteca para uma Faixa para atribuir.
  • Cada faixa tem controles de play/pause, volume e loop.

Salvar e Exportar

  • Salvar Projeto: salva no localStorage (apenas neste navegador).
  • Exportar JSON: baixa um arquivo .json com todo o projeto (inclui dataURLs se houver áudios locais).
  • Importar: carregue um JSON exportado para restaurar o projeto.

Temas

Use o botão no cabeçalho para alternar entre modo claro e escuro.
